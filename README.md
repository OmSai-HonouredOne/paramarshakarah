A comprehensive AI-driven career counseling system designed to bridge the gap between current skills and professional goals. **Paramarshakarah** (The Counselors) leverages advanced language models to provide personalized career roadmaps and educational guidance.

**Live Demo:** [https://paramarshakarah.onrender.com/](https://paramarshakarah.onrender.com/)

## üöÄ Key Features

- **Skill Analysis:** Deep analysis of user skills and target job roles.
- **Gap Identification:** Precise detection of skill gaps between the user's current profile and market requirements.
- **Smart Roadmaps:** Generation of step-by-step career progression paths.
- **Course Recommendations:** Curated lists of effective courses to help users upskill efficiently.

- **Backend:** Flask (Python)

## Security & Production Practices
- Environment variables used for sensitive data
- No credentials exposed in repository
- Flash messaging for user feedback
